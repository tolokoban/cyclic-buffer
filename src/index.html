The following performance tests have been performed on a AMD Phenom(tm) II X4 840 Processor running Linux version 3.13.0-24-generic. The browsers we used are:

  • Firefox 45.0
  • Google Chrome 50.0.2661.86 (64-bit)

Test 1

We push 2048 numbers, then we pop 2048 numbers. Then we repeat this process 10'000 times.

{ "Firefox, array": 114, "Firefox, buffer": 212, "Chrome, array": 171, "Chrome, buffer": 147 }

Test 2

We push 2048 strings, then we pop 2048 strings. Then we repeat this process 10'000 times.

{ "Firefox, array": 1897, "Firefox, buffer": 1955, "Chrome, array": 864, "Chrome, buffer": 836 }
